# Whack-A-Mole VR

Whack-A-Mole VR is a simple virtual reality game developed in **Unity** for **Meta Quest**.  
The player grabs a hammer and physically hits moles as they pop out of the table.

---

## Gameplay
- The game lasts **30 seconds**
- Two types of moles:
  - Normal mole: **1 hit**
  - Hard hat mole: **2 hits**
- Bomb mole: **ends the game immediately**
- The game becomes harder over time

- Each mole hit: **+0.5 seconds**

---

## Controls
- **Left controller joystick**: move closer to the table  
- **Right controller joystick**: rotate the camera  
- **Side button**:  
  - Press to start the game  
  - Press and hold to grab the hammer  

---

## How to Run
1. Open the project in **Unity**.
2. Select **Android** as the build platform.
3. Connect a **Meta Quest headset** using Link or Air Link.
4. Open the main scene.
5. Press **Play** or **Build & Run**.
6. Additionally, there is also a build under "Releases"
